# Style Bank

只在需要决定风格时读取本文件。

## 1. 黑白手绘讲解

适用：

- 概念解释
- 流程讲解
- 模块说明
- 场景化知识视频

特征：

- 黑白高对比
- 粗线条
- 简笔漫画/手绘说明书感
- 人物、对白框、板书、桌面、电脑、书架等元素

中文提示词骨架：

```text
黑白手绘讲解风格，粗黑线条，白底黑线，高对比，像知识讲解短视频插画，一位简笔人物正在解释主题，画面干净，信息集中，16:9 横版，适合后期加字幕，图内不要出现密集长文本
```

英文提示词骨架：

```text
black and white hand-drawn explainer illustration, bold black outlines, clean white background, simple cartoon character explaining a concept, educational short-video style, clear composition, high contrast, 16:9, leave room for subtitles, no dense paragraphs
```

## 2. 黑底高对比标题卡

适用：

- 关键词
- 结论
- 定义
- 一眼记住的核心句

特征：

- 黑底
- 中心大字
- 强对比
- 可带少量霓虹或发光感
- 画面元素极少

中文提示词骨架：

```text
黑底高对比标题卡风格，中央视觉强烈，少量大字，科技感或霓虹发光感，画面极简，适合强调关键词，16:9 横版，不要复杂背景，不要密集文本
```

英文提示词骨架：

```text
high-contrast black title card, minimal composition, a few large words only, subtle neon glow, strong central focus, cinematic educational card, 16:9, no clutter, no dense text
```

## 3. 记忆板书 / 口诀图

适用：

- 步骤
- 箭头链路
- 口诀
- 顺序关系

特征：

- 像白板、黑板、板书、便签图
- 可以有箭头、圈注、手写字感
- 图内字要短

中文提示词骨架：

```text
记忆板书风格，像课堂白板或黑板，少量大字，箭头连接，手写感，步骤清晰，适合讲记忆口诀，16:9 横版，不要长段文字
```

英文提示词骨架：

```text
memory board style, like a classroom whiteboard or blackboard, a few large handwritten words, arrows connecting key steps, simple mnemonic visual, 16:9, no long paragraphs
```

## 4. 轻科技信息图

适用：

- 技术模块
- 系统结构
- 抽象概念
- 比较正式的科技主题

特征：

- 简化 UI
- 但不能做成复杂假后台
- 少量发光边框、卡片、图标

中文提示词骨架：

```text
轻科技信息图风格，简洁科技感界面，结构化信息块，少量图标和标题，画面干净，适合讲系统模块，16:9 横版，不要复杂假后台，不要密集小字
```

英文提示词骨架：

```text
lightweight tech infographic style, clean structured interface, a few simple info blocks and icons, modern educational visual, 16:9, avoid dense small text, avoid messy fake dashboard
```

## 统一负面限制词

可以按需组合：

```text
watermark, logo, signature, dense text, wrong text, gibberish text, blurry face, broken hands, extra fingers, deformed body, low contrast, cluttered layout, cropped head, messy composition
```
